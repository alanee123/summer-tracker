# PRD: Летний трекер задач и занятий

**Версия:** 1.1  
**Дата:** 25 июня 2026  
**Статус:** Готово к разработке  
**Изменения v1.1:** добавлен раздел «Уроки» (F-50–F-56)

---

## 1. Обзор

Летний трекер — веб-приложение для подростков (14–17 лет), помогающее структурировать лето: выполнять ежедневные задачи, отслеживать занятия и хобби, достигать больших целей. Приложение мотивирует через визуальный прогресс и серию дней подряд.

**Целевая аудитория:** подростки 14–17 лет и их родители.  
**Платформа:** браузер (HTML/CSS/JS, без бэкенда), данные хранятся в `localStorage`.

---

## 2. Проблема

Летом у подростков нет внешней структуры (школы), и продуктивное время легко теряется. Нет удобного инструмента, который одновременно: отслеживает ежедневные обязанности, логирует занятия и хобби, помогает двигаться к большим целям и держит в голове расписание уроков и репетиторства.

---

## 3. Цели продукта

- Помочь подростку самостоятельно организовать лето без родительского контроля.
- Создать позитивную петлю мотивации через серии дней (streak) и визуальный прогресс.
- Снизить трение до минимума: открыл файл — сразу видишь, что делать сегодня.

---

## 4. Не входит в скоуп (v1)

- Синхронизация между устройствами / облако
- Уведомления и напоминания
- Многопользовательский режим / аккаунты
- Мобильное приложение

---

## 5. Функциональные требования

### 5.1 Главная страница

| # | Требование |
|---|------------|
| F-01 | Отображать текущую дату и день недели |
| F-02 | Показывать прогресс-бар лета (1 июня – 31 августа) с количеством оставшихся дней |
| F-03 | Показывать 4 ключевых метрики: задач выполнено сегодня, серия дней подряд, целей выполнено, занятий записано всего |
| F-04 | Список невыполненных задач на сегодня с возможностью перейти к ним |

### 5.2 Задачи на день

| # | Требование |
|---|------------|
| F-10 | Список повторяющихся ежедневных задач с эмодзи и категорией |
| F-11 | Отметка выполнения по нажатию — задача зачёркивается и выцветает |
| F-12 | Прогресс-бар дня: `выполнено / всего` |
| F-13 | Навигация по дням (вперёд/назад) с кнопкой «Сегодня» |
| F-14 | Добавление задачи: название, эмодзи, категория |
| F-15 | Удаление задачи — убирает её из всех дней |
| F-16 | Категории задач: Спорт, Учёба, Здоровье, Хобби, Другое |

### 5.3 Занятия

| # | Требование |
|---|------------|
| F-20 | Карточки занятий: название, эмодзи, категория, суммарный счётчик, единица измерения |
| F-21 | Кнопка «Записать занятие» — запрашивает значение (по умолчанию 1) и сохраняет с датой |
| F-22 | Визуализация активности: точки за последние 30 дней (закрашена = было занятие) |
| F-23 | Добавление занятия: название, эмодзи, категория, единица измерения |
| F-24 | Редактирование и удаление занятия |

### 5.4 Цели на лето

| # | Требование |
|---|------------|
| F-30 | Карточка цели: название, текущее / целевое значение, единица, цветовой прогресс-бар, процент выполнения |
| F-31 | Быстрые кнопки увеличения прогресса: +1, +5, произвольное значение |
| F-32 | Кнопка −1 для корректировки |
| F-33 | Бейдж «Выполнено!» и подсветка карточки при достижении цели |
| F-34 | Добавление цели: название, целевое число, единица, начальное значение, цвет из палитры (6 цветов) |
| F-35 | Редактирование и удаление цели |

### 5.5 Уроки

| # | Требование |
|---|------------|
| F-50 | Недельная сетка (Пн–Вс): для каждого дня отображаются уроки, назначенные на этот день недели |
| F-51 | Карточка урока в сетке показывает: время начала, название, преподавателя (если указан), длительность |
| F-52 | Клик по карточке отмечает урок выполненным (галочка + приглушённый вид); повторный клик снимает отметку |
| F-53 | Навигация по неделям (назад/вперёд) с кнопкой «Эта неделя» |
| F-54 | Список всех уроков с отображением дней недели в виде таблетки (активные подсвечены акцентным цветом) |
| F-55 | Добавление урока: эмодзи, предмет/название, преподаватель (необязательно), время начала, длительность в минутах, дни недели (мультивыбор Пн–Вс), цвет из палитры (8 цветов) |
| F-56 | Редактирование и удаление урока |

### 5.6 Серия дней (Streak)

| # | Требование |
|---|------------|
| F-40 | Серия считается за дни, когда выполнены все задачи |
| F-41 | Если сегодня ещё не выполнено ни одной задачи — день не учитывается (серия не обрывается) |
| F-42 | Счётчик серии отображается в сайдбаре и на главной |

> Примечание: уроки не учитываются в streak — их пропуск не обрывает серию.

## 7. Архитектура данных

```json
{
  "tasks": [
    { "id": "string", "name": "string", "emoji": "string", "category": "sport|study|health|hobby|other" }
  ],
  "checks": {
    "YYYY-MM-DD": ["taskId", "taskId"]
  },
  "activities": [
    {
      "id": "string",
      "name": "string",
      "emoji": "string",
      "category": "string",
      "unit": "string",
      "sessions": [{ "date": "YYYY-MM-DD", "value": 1 }]
    }
  ],
  "goals": [
    { "id": "string", "name": "string", "target": 10, "current": 2, "unit": "string", "color": "#hex" }
  ],
  "lessons": [
    {
      "id": "string",
      "subject": "string",
      "emoji": "string",
      "teacher": "string",
      "time": "HH:MM",
      "duration": 60,
      "days": [0, 2, 4],
      "color": "#hex"
    }
  ],
  "lessonChecks": {
    "YYYY-MM-DD": ["lessonId", "lessonId"]
  }
}
```

`days` — массив индексов дней недели с понедельника: 0 = Пн, 1 = Вт, …, 6 = Вс.

Ключ в `localStorage`: `summer_tracker`.

## 8. UX-решения

**Навигация:** фиксированный левый сайдбар с 5 разделами. Активный раздел подсвечивается акцентным цветом.

**Первый запуск:** при пустом хранилище автоматически создаются 3 примера задач, 2 занятия и 3 цели — пользователь сразу видит заполненный интерфейс.

**Цветовой язык категорий:**

| Категория | Цвет |
|-----------|------|
| Спорт     | Голубой `#38bdf8` |
| Учёба     | Фиолетовый `#7c6fff` |
| Здоровье  | Зелёный `#4ade80` |
| Хобби     | Розовый `#ff6b9d` |
| Другое    | Жёлтый `#fbbf24` |

**Модальные окна:** открываются по действию пользователя, закрываются по клику на оверлей или Esc.

## 9. Метрики успеха

| Метрика | Цель |
|---------|------|
| Средняя серия дней к концу лета | ≥ 14 дней |
| Доля целей, достигнутых к 31 августа | ≥ 50% |
| Задач выполнено в % за лето | ≥ 70% дней |

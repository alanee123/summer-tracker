# ☀️ Летний трекер задач и занятий

Веб-приложение для подростков (14–17 лет), помогающее структурировать лето: выполнять
ежедневные задачи, отслеживать занятия и хобби, достигать больших целей и держать в голове
расписание уроков. Мотивация — через визуальный прогресс и серию дней подряд (streak).

Всё приложение — **один файл `index.html`**, работает офлайн, без бэкенда. Данные хранятся
в `localStorage` браузера.

## ✨ Возможности

- **🏠 Главная** — текущая дата, прогресс-бар лета (1 июня – 31 августа) с оставшимися днями,
  4 ключевые метрики и список невыполненных задач на сегодня.
- **✅ Задачи** — повторяющиеся ежедневные дела с эмодзи и категориями, отметка выполнения,
  прогресс-бар дня, навигация по дням.
- **📊 Занятия** — карточки активностей с суммарным счётчиком и единицей измерения, запись
  занятий, визуализация активности за последние 30 дней.
- **🎯 Цели** — цветные прогресс-бары, быстрые кнопки +1/+5/−1/произвольно, бейдж «Выполнено!».
- **📚 Уроки** — недельная сетка Пн–Вс, отметка проведённых уроков, список с днями-таблетками.
- **🔔 Напоминания** — баннер и браузерное уведомление за заданное время до начала урока
  (работают, пока вкладка открыта).
- **🔥 Серия дней** — считается за дни, когда выполнены все задачи; незавершённый сегодняшний
  день серию не обрывает.
- **📱 Адаптивность** — на телефоне нижняя навигация (таб-бар), компактный верхний хедер
  с серией и одноколоночная раскладка.

## 🚀 Как запустить

Просто откройте `index.html` в браузере (двойной клик) — интернет не нужен.

При первом запуске автоматически создаются примеры (3 задачи, 2 занятия, 3 цели), чтобы сразу
видеть заполненный интерфейс.

## 📲 Поделиться

Файл [`whatsapp-share.html`](whatsapp-share.html) собирает ссылку `wa.me` с готовым сообщением
и QR-кодом для шеринга приложения в WhatsApp. QR генерируется офлайн, без сторонних сервисов.

## 🛠 Технологии

Чистые HTML + CSS + JavaScript, без зависимостей и сборки. Тёмная тема, хранение в
`localStorage` (ключ `summer_tracker`).

## ⚠️ Ограничения (v1)

- Нет синхронизации между устройствами / облака.
- Напоминания работают только пока вкладка открыта (для пушей при закрытом приложении нужен
  бэкенд с Web Push или нативное приложение).
- Один пользователь, без аккаунтов.

Полная спецификация — в [`prd.md`](prd.md).
